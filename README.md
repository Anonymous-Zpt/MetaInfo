MetaInfo es una herramienta que nos permite extraer los metadatos de cualquier tipo de archivo:

- EXIF

- GPS

- IPTC

- XMP

- JFIF

- GeoTIFF

- Perfil ICC

- Photoshop IRB

- FlashPix

- AFCP e ID3.
